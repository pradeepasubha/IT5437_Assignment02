## How to Run
1. Clone the repository
2. Open `homography_orb_matching.py`
3. Run the script:
   python homography_orb_matching.py

## Usage
- When Image 1 window opens, click 6 distinctive points on the circuit board
- Press any key to continue
- When Image 2 window opens, click the same 6 corresponding points
- Press any key to continue
- Results will be displayed automatically

## Notes
- Images c1.jpg and c2.jpg are included in the repository
- No need to change file paths as the code reads from the same directory

## Update File Paths in Code
Before uploading, change the hardcoded paths to relative paths so anyone can run it.
